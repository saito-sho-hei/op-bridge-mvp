# 営業利益ブリッジ (OP Bridge) MVP

## 概要
月次データの数値を入力するだけで、**「営業利益の増減理由（ブリッジ）」と「PL比較表」** のA4レポート（PDF）を即座に生成するツールです。

## 使い方

1. **起動**:
   ```bash
   cd op-bridge-mvp
   npm run dev
   ```
   ブラウザで `http://localhost:3000` を開きます。

2. **入力**:
   - **対象月・拠点名**: 必須です。
   - **比較モード**: 「前月比」または「目標比」を選びます。
   - **数値入力**: 単位は「千円」です。
     - カンマ付き入力OK（例: 12,000）
     - 任意項目（消耗品、輸送費など）は空欄でもOK（自動で0になります）
     - 必須項目（売上、材料、労務、販管）が空欄だとエラーになります。

3. **出力**:
   - 「PDFを作成する」ボタンを押すと、計算結果がPDFファイルとしてダウンロードされます。
   - ページ1: PLサマリー（前月差・増減率）
   - ページ2: ウォーターフォールチャート（要因分析コメント付き）

## 技術スタック
- Framework: Next.js 14 (App Router)
- CSS: TailwindCSS
- PDF: @react-pdf/renderer
- Logic: TypeScript

## 拡張ポイント（将来のSaaS化に向けて）
- **認証**: ユーザーごとの保存機能はありません。SaaS化時はAuthが必要です。
- **DB連携**: 現状は入力値を保持しません。API経由でDBに保存する機能を推奨。
- **疑似目標**: 過去平均の自動算出は未実装です。
